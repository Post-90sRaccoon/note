# 正则表达式

### creating a regular expression 

```javascript
var re1 = new RegExp("abc")
var re2 = /abc/

re1 == re2
//false

var a = new RegExp('\\a')
//  \a
a2 = /\\a/
// \\a
a3 = new RegExp(String.fromCharCode(92, 97))
// \a

//表示/a\/bc/
a4 = new RegExp(String.fromCharCode(97, 92, 47, 98, 99))
a5 = new RegExp('a\\/bc')
a6 = new RegExp('\\n')
// \n 

\s 就是s
\k 就是k  被忽略掉

/a+/
//连续出现字母a多于一次

/a\+/ 
//表示一个a和加号

/a\+/.test('wioefa+joisf')
//true
```
